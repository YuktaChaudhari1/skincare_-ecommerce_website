# 🌸 Aurielle – Authentication System (MERN)

Aurielle is a luxury skincare brand web application built using the MERN stack.

This project currently focuses on implementing secure user authentication and MongoDB database integration.

---

## 🚀 Tech Stack

### Frontend
- React JS (Vite)
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt for password hashing

---

## ✨ Features Implemented

- User Registration (Sign Up)
- User Login (Sign In)
- Password Hashing using bcrypt
- JWT Token Generation
- Protected Routes
- MongoDB Database Connection
- Environment Variable Configuration
- Clean Folder Structure (Frontend & Backend separated)
